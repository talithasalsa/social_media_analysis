## Social Media Promotion Analysis

This project analyzes the digital promotion strategy of MOVA, a cashback application brand, across multiple social media platforms. The analysis explores how promotional content is distributed, what content formats perform best, and how messaging is used to attract users.
The project combines data collection, exploratory data analysis, and visualization to better understand how MOVA leverages social media for marketing and user acquisition.

📌 Project Objectives

The main goal of this project is to understand MOVA’s digital promotion strategy across social media platforms.

Key objectives include:
- Analyzing the distribution of promotional posts across platforms
- Examining content format preferences (video vs image/text)
- Comparing engagement performance between platforms
- Identifying patterns in promotional messaging
- Investigating how users and affiliates contribute to promotion activity

🌐 Platforms Analyzed

The study focuses on five major social media platforms:
- Instagram
- TikTok
- Facebook
- Twitter
- YouTube Shorts

A total of 100 MOVA-related promotional posts were collected and analyzed.

🔎 Methodology
1. Data Collection
  MOVA promotional posts were manually collected from the five social media platforms. Posts were selected based on whether they contained promotional messaging   related to the MOVA cashback application.

2. Data Processing
  The collected data was organized and cleaned using Python tools such as:
  - Pandas for data manipulation
  - Matplotlib / Seaborn for visualization
  - WordCloud for text analysis

3. Analysis Focus
The analysis explores several dimensions:
  - Platform distribution
  - Content format trends
  - Engagement performance
  - Promotional language patterns
  - Posting behavior across accounts
